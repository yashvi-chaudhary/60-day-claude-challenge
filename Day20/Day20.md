🚀 Day 20/60 – Building an AI-Powered Face Puzzle Game 🎮

📸 Today's challenge pushed AI-assisted front-end development to the next level by creating a fully functional Face Puzzle Game using a single HTML file.

The application includes:
✨ Live webcam access using the MediaDevices API 
📷 Face photo capture 
🧩 Multiple puzzle difficulties (3×3, 4×4, 5×5) 
🖱️ Drag-and-drop gameplay with touch support 
⏱️ Real-time timer and move counter 
✅ Correct piece detection 
🏆Win detection with a results screen 
💾 Local leaderboard using localStorage 
📱 Responsive design for desktop and mobile

One interesting part of this challenge was seeing how AI can generate not only the interface but also the game logic—from handling camera permissions and image slicing to implementing drag-and-drop interactions, timers, and persistent score tracking.
