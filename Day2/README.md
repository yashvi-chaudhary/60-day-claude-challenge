# Day 2 - Prompt Engineering

## Weak Prompt

**Prompt:**
"Tell me about marketing"

**Output:**
"Marketing is the process of promoting products and services to customers..."

### Issues

* No specific goal
* No audience defined
* No format requested
* No role assigned

---

## Engineered Prompt

**Prompt:**
"You are a marketing expert. List 5 social media tips for a small bakery. Use bullet points. Keep each under 2 sentences."

**Output:**

* Post behind-the-scenes baking reels.
* Share customer testimonials.
* Highlight daily specials.
* Use local hashtags.
* Run seasonal promotions.

### Improvements

* Clear goal defined
* Audience specified
* Format requested
* Length constrained
* Role assigned

## Key Takeaway

Better prompts produce more focused, actionable, and useful AI responses.

**Golden Formula:**
Role + Task + Context + Format + Constraints = Better Output
