# Day 3 - Role-Based Prompting

## Overview

Today's challenge focused on **Role-Based Prompting**, a technique where the AI is assigned a specific role before answering a question.

## Personas Used

* Everyday User
* Founder
* Developer

## Key Learning

The same question can produce completely different answers depending on the role assigned to the AI.

* **Everyday User:** Focuses on simplicity and clarity.
* **Founder:** Focuses on business value, leverage, and decision-making.
* **Developer:** Focuses on technical accuracy and implementation details.

## Main Takeaway

Role-based prompting helps generate more relevant and specialized responses by guiding the AI to think from a particular perspective.

## Universal Formula

**Act as [Role] + Audience: [Who] + Constraint: [Limit] + Task: [What]**

## Conclusion

By changing the AI's role, we can obtain answers tailored to different audiences and objectives, making prompt engineering more effective and purposeful.
